# Projeto MVC - Cadastro de Advogados

## Tecnologias

* ASP.NET MVC 5
* Entity Framework 6
* jQuery + Mask
* Bootstrap

## Como executar

1. Abrir a solução no Visual Studio
2. Executar (F5)

## Banco de dados

* LocalDB (.mdf)
* Criação automática

## Funcionalidades

* CRUD de Advogados
* Validação de campos
* Integração com ViaCEP
